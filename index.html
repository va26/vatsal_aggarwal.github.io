Secure and Autonomous UAV Navigation

I am working on a project to develop a secure and autonomous UAV navigation system using Fully Homomorphic Encryption (FHE). FHE is a powerful encryption scheme that allows us to perform computations on encrypted data without decrypting it first. This makes it ideal for applications where security is paramount, such as autonomous UAV navigation.


The current state-of-the-art in autonomous UAV navigation relies on machine learning (ML) techniques. However, these techniques are vulnerable to a variety of attacks, such as adversarial attacks and data poisoning. FHE can help to address these security risks by allowing us to train and deploy ML models on encrypted data. In my project, I am developing a novel FHE-based framework for autonomous UAV navigation.


The framework will be able to learn and adapt to its environment in real time, while still maintaining its security. This will make it possible to deploy autonomous UAVs in a variety of sensitive and critical applications, such as military surveillance and disaster response.

Hair Cell Counting

We are developing a complete pipeline to automate the counting of hair cells from inner ear image data. This system is especially useful in cases where there are few or no labels, and it works well without human annotations. Our pipeline consists of two steps:

  1. Segmentation: We use a machine learning model to generate segmented masks that isolate the region of interest (ROI) in the image. This ROI contains the hair cells and other structures in the inner ear.
  2. Object detection: We then use object detection to identify and count the hair cells in the ROI. This is done by identifying the individual hair cells and their features, such as their size and shape.

Our system has been shown to be accurate and efficient, and it can be used to count hair cells in a variety of inner ear image data. This information can be used to study the effects of hearing loss and other conditions on the inner ear.


A Two-Stage Hybrid Methodology for Forgery Detection And Palmprint Authentication

Biometric recognition is a rapidly growing field in artificial intelligence. It uses unique physical or behavioral traits to identify individuals. Some common biometrics include fingerprints, facial recognition, and iris scanning. While biometric recognition systems are becoming increasingly sophisticated, they are still vulnerable to spoofing attacks.


This project addresses the problem of spoofing attacks in palmprint recognition. We propose a two-step approach that first detects forgery attacks and then authenticates the images. For forgery detection, we extract three types of features: ELA (Error Level Analysis), DFT (Discrete Fourier Transform), and original image features. We train three basic CNN models on these features and achieve the highest accuracy of 99.6% for DFT features. For image authentication, we use ResNet-50, MobileNet-v3, and Inception-v3 and achieve the highest accuracy of 100% for all three models using DFT features.
